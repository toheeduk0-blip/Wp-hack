How to Use Your Access Keys

To send a message, make a POST request to your project's API endpoint. This endpoint securely handles your credentials.

API Endpoint URL

Your API endpoint is: /api/send
This URL is generated automatically based on your current window location.

Request Body

Your POST request must be in JSON format and include:

  • accessKey: The key generated from this panel.
  • message: The text message you want to send.

JavaScript (Fetch API) Example:


const apiKey = 'YOUR_GENERATED_ACCESS_KEY';
const messageText = 'Hello from my application!';
const apiUrl = '/api/send';

fetch(apiUrl, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        accessKey: apiKey,
        message: messageText,
    }),
})
.then(response => response.json())
.then(data => console.log('Success:', data))
.catch(error => console.error('Error:', error));

Python (requests) Example:


import requests
import json

api_key = "YOUR_GENERATED_ACCESS_KEY"
message_text = "Hello from my Python script!"
api_url = "/api/send"

payload = { "accessKey": api_key, "message": message_text }

try:
    response = requests.post(api_url, json=payload)
    response.raise_for_status()
    print("Success:", response.json())
except requests.exceptions.RequestException as e:
    print("Error:", e)
